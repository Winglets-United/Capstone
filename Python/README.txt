This folder contains python code for creating VORLAX input files and visualizing vorlax log data. 
This code will NOT be kept up-to-date, as the primary repo for the Vorlax_Lib modules can be found at: https://github.com/tucono/VorlaxParser_Py/

The python code included REQUIRES the following:
64-bit Python 3.x
 - Libraries:
  - NumPy
  - MayAvi
  - VTK
  - PyQt5
Visual C Compiler

For Python and Numpy, it is recommended to use Anaconda 64-bit as it bundles many popular libraries with python.
